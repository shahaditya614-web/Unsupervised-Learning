# 📊 Unsupervised Learning using K-Means Clustering

A beginner-friendly implementation of **Unsupervised Machine Learning** using the **K-Means Clustering Algorithm** in **Python**. This project demonstrates how to perform **Customer Segmentation** by grouping unlabeled data into meaningful clusters using **Scikit-learn**.

---

# 📌 Project Overview

This project demonstrates the concepts of **Unsupervised Learning**, where the dataset has **no target labels**. The notebook applies the **K-Means Clustering** algorithm to perform **Customer Segmentation** based on customer characteristics.

The notebook is designed for **Google Colab** and is suitable for beginners learning Machine Learning and Data Science.
